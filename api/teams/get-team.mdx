## Get Team

Retrieve detailed information about a specific team, including team members.

### Endpoint

```
GET /api/teams/{teamId}
```

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Path Parameters

<ParamField path="teamId" type="string" required>
  Team UUID
</ParamField>

### Response

<ResponseField name="id" type="string">
  Team UUID
</ResponseField>

<ResponseField name="name" type="string">
  Team name
</ResponseField>

<ResponseField name="accessCode" type="string">
  Unique access code for joining the team
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of creation
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

<ResponseField name="members" type="array">
  Array of team member objects
  
  <Expandable title="member object">
    <ResponseField name="id" type="string">
      Team user relationship UUID
    </ResponseField>
    
    <ResponseField name="userId" type="string">
      User UUID
    </ResponseField>
    
    <ResponseField name="teamId" type="string">
      Team UUID
    </ResponseField>
    
    <ResponseField name="role" type="string">
      Member role: `team-member`, `team-view-only`, or `team-manager`
    </ResponseField>
    
    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp when user joined team
    </ResponseField>
    
    <ResponseField name="user" type="object">
      User information
      
      <Expandable title="user object">
        <ResponseField name="id" type="string">
          User UUID
        </ResponseField>
        
        <ResponseField name="username" type="string">
          Username
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>

```bash curl
curl https://your-umami-instance.com/api/teams/660e8400-e29b-41d4-a716-446655440001 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const teamId = '660e8400-e29b-41d4-a716-446655440001';

const response = await fetch(
  `https://your-umami-instance.com/api/teams/${teamId}`,
  {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN',
    },
  }
);

const team = await response.json();
```

```python Python
import requests

team_id = '660e8400-e29b-41d4-a716-446655440001'

response = requests.get(
    f'https://your-umami-instance.com/api/teams/{team_id}',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)

team = response.json()
```

</CodeGroup>

### Error Responses

<ResponseField name="401" type="error">
  Unauthorized - You must be a member of this team
</ResponseField>

<ResponseField name="404" type="error">
  Not Found - Team does not exist
</ResponseField>

<Note>
  You must be a member of the team to view its details. The response includes full member information for team management.
</Note>
