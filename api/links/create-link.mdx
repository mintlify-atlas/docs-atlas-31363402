## Create Link

Create a new short link for URL shortening with integrated analytics tracking.

### Endpoint

```
POST /api/links
```

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Request Body

<ParamField body="name" type="string" required>
  Link name (max 100 characters)
</ParamField>

<ParamField body="url" type="string" required>
  Destination URL (max 500 characters)
</ParamField>

<ParamField body="slug" type="string" required>
  Unique URL slug (max 100 characters). Must be unique across all links.
</ParamField>

<ParamField body="teamId" type="string">
  Team UUID if creating a team link. Omit for personal link.
</ParamField>

<ParamField body="id" type="string">
  Optional custom UUID for the link. Generated automatically if not provided.
</ParamField>

### Response

<ResponseField name="id" type="string">
  Link UUID
</ResponseField>

<ResponseField name="name" type="string">
  Link name
</ResponseField>

<ResponseField name="url" type="string">
  Destination URL
</ResponseField>

<ResponseField name="slug" type="string">
  URL slug
</ResponseField>

<ResponseField name="userId" type="string">
  Owner user UUID (null for team links)
</ResponseField>

<ResponseField name="teamId" type="string">
  Team UUID (null for personal links)
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of creation
</ResponseField>

### Example Request

<CodeGroup>

```bash curl
curl -X POST "https://your-umami-instance.com/api/links" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Product Launch",
    "url": "https://example.com/product-launch",
    "slug": "launch2024"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-umami-instance.com/api/links', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    name: 'Product Launch',
    url: 'https://example.com/product-launch',
    slug: 'launch2024',
  }),
});

const link = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://your-umami-instance.com/api/links',
    headers={'Authorization': 'Bearer YOUR_TOKEN'},
    json={
        'name': 'Product Launch',
        'url': 'https://example.com/product-launch',
        'slug': 'launch2024'
    }
)

link = response.json()
```

</CodeGroup>

## Creating a Team Link

To create a link owned by a team, include the `teamId` parameter:

```bash curl
curl -X POST "https://your-umami-instance.com/api/links" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Marketing Campaign",
    "url": "https://example.com/campaign",
    "slug": "campaign-q1",
    "teamId": "880e8400-e29b-41d4-a716-446655440003"
  }'
```

<Note>
  You must have permission to create websites for the team. The slug must be unique across all links in the system.
</Note>
