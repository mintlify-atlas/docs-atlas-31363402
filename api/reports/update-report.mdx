## Update Report

Update the configuration of an existing report.

### Endpoint

```
POST /api/reports/{reportId}
```

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Path Parameters

<ParamField path="reportId" type="string" required>
  Report UUID
</ParamField>

### Request Body

<ParamField body="websiteId" type="string" required>
  Website UUID
</ParamField>

<ParamField body="type" type="string" required>
  Report type (cannot be changed to a different type)
</ParamField>

<ParamField body="name" type="string" required>
  Updated report name (max 200 characters)
</ParamField>

<ParamField body="description" type="string">
  Updated report description (max 500 characters)
</ParamField>

<ParamField body="parameters" type="object" required>
  Updated report-specific parameters
</ParamField>

<Note>
  All fields must be provided, even if only updating some values. The report type-specific parameter requirements must be met.
</Note>

### Response

<ResponseField name="id" type="string">
  Report UUID
</ResponseField>

<ResponseField name="websiteId" type="string">
  Website UUID
</ResponseField>

<ResponseField name="userId" type="string">
  User who last updated the report
</ResponseField>

<ResponseField name="type" type="string">
  Report type
</ResponseField>

<ResponseField name="name" type="string">
  Updated report name
</ResponseField>

<ResponseField name="description" type="string">
  Updated report description
</ResponseField>

<ResponseField name="parameters" type="object">
  Updated report parameters
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of update
</ResponseField>
