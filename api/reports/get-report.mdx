## Get Report

Retrieve detailed information about a specific report.

### Endpoint

```
GET /api/reports/{reportId}
```

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Path Parameters

<ParamField path="reportId" type="string" required>
  Report UUID
</ParamField>

### Response

<ResponseField name="id" type="string">
  Report UUID
</ResponseField>

<ResponseField name="websiteId" type="string">
  Website UUID
</ResponseField>

<ResponseField name="userId" type="string">
  User who created the report
</ResponseField>

<ResponseField name="type" type="string">
  Report type
</ResponseField>

<ResponseField name="name" type="string">
  Report name
</ResponseField>

<ResponseField name="description" type="string">
  Report description
</ResponseField>

<ResponseField name="parameters" type="object">
  Report-specific parameters
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of creation
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of last update
</ResponseField>

### Example Request

<CodeGroup>

```bash curl
curl https://your-umami-instance.com/api/reports/dd0e8400-e29b-41d4-a716-446655440008 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const reportId = 'dd0e8400-e29b-41d4-a716-446655440008';

const response = await fetch(
  `https://your-umami-instance.com/api/reports/${reportId}`,
  {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN',
    },
  }
);

const report = await response.json();
```

```python Python
import requests

report_id = 'dd0e8400-e29b-41d4-a716-446655440008'

response = requests.get(
    f'https://your-umami-instance.com/api/reports/{report_id}',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)

report = response.json()
```

</CodeGroup>

### Error Responses

<ResponseField name="401" type="error">
  Unauthorized - You don't have permission to view this report
</ResponseField>

<ResponseField name="404" type="error">
  Not Found - Report does not exist
</ResponseField>

<Note>
  You must have view permissions for the website that owns this report.
</Note>
